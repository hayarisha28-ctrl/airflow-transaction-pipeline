# 🚀 Airflow Transaction Pipeline

## 📌 Description
This project uses Apache Airflow to process transactions and generate a report.

## ⚙️ Tools
- Airflow
- Docker
- Python

## 🔄 Steps
1. Start
2. Process transactions
3. Run script
4. Read report
5. Send email

## 📊 Output
File created:
/data/transactions_report.txt

## 🚀 Run
docker compose -f airflow.yaml up -d

Open:
http://localhost:8089
